## Packages
framer-motion | Page transitions, AI analysis sequence, and smooth layout animations
lucide-react | Premium scalable icons
clsx | Conditional class merging
tailwind-merge | Tailwind class deduplication

## Notes
- This is a frontend-only mockup. No API integrations required.
- All views are orchestrated via local React state as requested (no routing library needed).
- Custom fonts (Syne, DM Sans, IBM Plex Mono) are loaded via Google Fonts in index.css.
- The UI heavily relies on customized CSS variables and Tailwind arbitrary values for exact color matching.
