# Courtside — Refinement Prompt (V2)

Apply the following refinements to the existing Courtside codebase. These changes affect the design system, the home page layout, and add a new polling/leaderboard feature.

## 1. Design System Overhaul — ESPN/NBA.com Alignment

The current design feels too startup-y and light. Shift it to feel more like ESPN's NBA page and NBA.com — authoritative, sports-native, and information-dense. Here are the specific changes:

### Typography
Replace the current fonts entirely:

- **Headlines, navigation, bold labels:** Use `"Barlow Condensed"` (Google Fonts), weights 600 and 700. This is close to ESPN's condensed headline font. Uppercase where appropriate for section headers.
- **Body text, editorial content, descriptions:** Use `"Source Sans 3"` (Google Fonts), weights 400 and 600. Clean, highly readable, sports-broadcast feel.
- **All numbers, stats, scores, odds, data:** Use `"IBM Plex Mono"` (Google Fonts), weights 400, 500, 600. Monospace for all numerical data — non-negotiable.

Load all three from Google Fonts:
```
https://fonts.googleapis.com/css2?family=Barlow+Condensed:wght@400;500;600;700&family=Source+Sans+3:wght@300;400;500;600;700&family=IBM+Plex+Mono:wght@400;500;600&display=swap
```

### Color Palette
Shift from the warm off-white palette to something closer to ESPN/NBA.com — a crisper, more sports-media feel:

```
Background:         #FFFFFF  (pure white — like ESPN)
Surface/Sections:   #F5F5F5  (light gray section backgrounds)
Card:               #FFFFFF  (white cards with border)
Dark sections:      #111111  (near black — for featured hero areas, player profiles)
Dark card:          #1A1A1A  (cards on dark background)

Text Primary:       #111111  (near black)
Text Secondary:     #444444  (dark gray)
Text Tertiary:      #888888  (medium gray)
Text Muted:         #AAAAAA  (light gray)
Text on Dark:       #FFFFFF  (white on dark sections)
Text on Dark Muted: #999999  (gray on dark sections)

Border:             #E0E0E0  (standard divider)
Border Strong:      #CCCCCC  (emphasized divider)

Accent Blue:        #1D428A  (NBA official blue — use for primary CTA, links, active states)
Accent Red:         #C8102E  (NBA official red — use for live indicators, alerts, losses)
Win Green:          #008248  (slightly darker green — wins, positive signals)
Warning Amber:      #F5A623  (amber — medium confidence, caution)

Live Pulse:         #C8102E  (red pulse for live games — like ESPN's live indicator)
```

### Layout Principles
- **Full-width sections** alternating between white and light gray backgrounds — like ESPN's modular layout
- **Section headers** in Barlow Condensed 700, uppercase, 13-14px, with a left-aligned red/blue accent bar (3px thick, 24px wide) to the left of the text — this is the ESPN signature pattern
- **Score ticker at very top** should have a dark background (#111111) with white text — matching ESPN's scoreboard bar
- **Cards** use subtle borders (1px #E0E0E0) with NO border-radius larger than 8px. ESPN and NBA.com use very minimal rounding.
- **Horizontal dividers** between content sections, not excessive whitespace
- **Dense information layout** — less padding, more content visible above the fold. ESPN packs information tight. Follow that density.

### Component Styling Updates
- **Buttons:** Primary buttons use NBA blue (#1D428A) background with white text. Secondary buttons use white background with #1D428A border and text.
- **Badges/Pills:** Confidence badges (HIGH/MED/COND) should be small, uppercase, Barlow Condensed 600, with subtle background color. Not rounded pills — more like rectangular tags with 4px border-radius.
- **Score displays:** Large, bold, IBM Plex Mono. Live scores should have the red live dot (pulsing) matching ESPN's style.
- **Team abbreviations:** Always Barlow Condensed 700, uppercase.
- **Section headers pattern:**
```html
<div style="display: flex; align-items: center; gap: 8px; margin-bottom: 12px;">
  <div style="width: 3px; height: 18px; background: #C8102E; border-radius: 1px;"></div>
  <h2 style="font-family: 'Barlow Condensed'; font-weight: 700; font-size: 14px; text-transform: uppercase; letter-spacing: 0.5px; color: #111;">Today's Hot Takes</h2>
</div>
```

## 3. Polling Feature — "Courtside Calls"

This is a new feature section that sits on the home page (between the game list and hot takes) AND has its own dedicated page at `/polls`.

### How It Works

1. **Users generate polls** by typing into the ask bar. When the analysis card appears, there's a prominent "Create a Courtside Call" button at the bottom.
2. Clicking it creates a **poll** that other users can vote on: AGREE (with Courtside's analysis) or FADE (take the other side).
3. The poll is tied to a real game outcome. When the game ends and the stat resolves, the poll auto-resolves.
4. Users who voted correctly earn **reputation points**:
   - Correct vote on a HIGH confidence call: +3 points
   - Correct vote on a MED confidence call: +5 points (harder to get right)
   - Correct vote on a COND confidence call: +8 points (highest reward for hardest calls)
   - Incorrect vote: 0 points (never negative — keep it fun)
   - Creating a poll that gets 10+ votes: +2 bonus points
5. Points accumulate on a **leaderboard**.

### Home Page: Active Polls Module

Place this between the game list and hot takes. Show 3-4 active polls in a compact format.

Each poll card:
- Left side: The proposition in Source Sans 3 600, 15px: "Brunson over 30 points vs BOS"
- Below: Courtside's verdict (e.g. "LEAN UNDER · 42%") in IBM Plex Mono, colored by verdict
- Center: Vote bar showing AGREE vs FADE percentages as a horizontal split bar (like ESPN's prediction polls)
  - AGREE side: NBA blue (#1D428A)
  - FADE side: gray (#888)
  - Show total vote count below
- Right side: Two voting buttons stacked: "AGREE" (blue outline) and "FADE" (gray outline)
- Bottom: Time until resolution: "Resolves at 10:30 PM ET" or "Resolves when BOS @ NYK ends"
- If resolved: show the result with a green check (correct side) or red X. Show points earned.

Section header: `COURTSIDE CALLS — LIVE` with the red accent bar

### Polls Page (`/polls`)

Full-page view of all polls.

**Tabs at top:** Active | Resolved | My Calls

**Active tab:** Grid/list of all open polls. Filterable by game, confidence level, stat type.

**Resolved tab:** Shows resolved polls with results, point awards, and accuracy tracking.

**My Calls tab:** User's personal polling history — their votes, outcomes, points earned, current streak.
